# 🌈 Weather App

A simple and modern weather application built with React that allows users to search for any city and view its current weather information instantly.
## ☀Live Demo
👉 https://react-weather-app-lyart-theta.vercel.app/
##  📌Project Description

This project is a responsive weather dashboard that fetches real-time weather data from an API and displays key information such as temperature, weather condition, humidity, and wind speed. It is designed to provide a clean and user-friendly experience for checking the weather quickly.

## 🛠Tech Stack

- React.js
- Vite
- JavaScript
- Axios
- Tailwind CSS
- OpenWeatherMap API

##  📂Project Structure

```bash
src/
  App.jsx
  components/
    SearchBar/
    WeatherDashboard/
      WeatherDetails/
      WeatherStats/
      WeatherSummary/
  hooks/
    useSearch.js
    useWeather.js
  services/
    baseUrl.js
    weatherApi.js
  utils/
    formatTemp.js
    getWeatherIcon.js
```

## ✨Features

- Search weather by city name
- Display current temperature and weather condition
- Show humidity and wind speed details
- Loading and error states for better user experience
- Clean, card-based UI with smooth visual styling

## 🚩Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
